# AI Procurement Intelligence System

## 🔗 Live App
https://ai-procurement-agent.streamlit.app/

## 📌 Overview
End-to-end AI system to analyze procurement data, detect risks, and generate insights via an interactive chat-based interface and dashboard.

## ⚙️ Features
- Spending & Contract Analysis
- Supplier Insights & Concentration
- Risk Detection (high-risk contracts)
- Forecasting trends
- Chat-based AI Agent (multi-intent)
- Interactive dashboard (Plotly + Streamlit)

## 🧠 Architecture
1. Data Pipeline (cleaning & standardization)
2. Analytics Engine (modular functions)
3. Agent System (intent detection + routing)
4. Streamlit UI (chat + dashboards)

## 🛠️ Tech Stack
Python, Pandas, Plotly, Streamlit

## ▶️ Run Locally

streamlit run app.py
